# These three packages share a tight compatibility matrix: aioboto3
# wraps aiobotocore, which wraps botocore — pin all three together to
# avoid resolver-induced surprises.
#
# IMPORTANT: aioboto3 15.x / aiobotocore 2.25 / boto3 1.40 do not exist

# supply-chain hazard — anyone who registers the missing version on
# PyPI gets their package installed in every build. Pinning to real
# released versions only. aioboto3 13.1.1 pins aiobotocore==2.13.1
# exactly, so we match — bumping either alone breaks `pip install`.
